# VDOT Traffic Cabinet MR Tool

#### Introduction

The VDOT Traffic Cabinet mixed reality Training Tool is a proof-of-concept training application in collaboration with the Virginia Department of Transportation (VDOT). The prototype simulates the internal layout and interactions of a real traffic signal control cabinet, allowing trainees to explore cabinet components in a safe and controlled environment. The application ultimately aims to demonstrate how XR technologies can support VDOT workforce training.

The application currently runs on Unity 6000 2.10f1:

- Apple Vision Pro (visionOS) using Unity PolySpatial
